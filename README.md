# 🌤 Weather App

A simple and dynamic weather application built with **Node.js**, **Express (HTTP server)**, and **OpenWeatherMap API**.  
It allows users to search for any city and instantly view real‑time weather details.

---

## 🚀 Features
- 🔍 Search weather by city name
- 🌡️ Displays temperature in Celsius
- ☁️ Shows weather condition/description
- 🎨 Attractive and responsive frontend (HTML + CSS + JS)
- ⚡ Fast API integration using **Axios**

---

## 🛠️ Tech Stack
- **Backend:** Node.js, HTTP module, Axios
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **API:** [OpenWeatherMap](https://openweathermap.org/api)
